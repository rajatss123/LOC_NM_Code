# LOC_NM_Code

Pilot study of responsive nucleus accumbens deep brain stimulation for Loss of Control eating (Shivacharan, Rolle, et al)

Repository containing code for the data presented in the above paper for: LOC Buffet, Ambulatory LFP Analysis, LFP Detection Analysis. All analysis was done in MATLAB 2014b and utilized FieldTrip Toolbox (https://www.fieldtriptoolbox.org/)
